# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar
   ```
   npm install
   ```
3. Tener el Nest CLI instalado
   ```
   npm i -g @nest/cli
   ```
4. Levantar la base de datos
   ```
   docker-compose up -d
   ```
